# Trabalho mensal 1BIM - Trabalho de Conclusão de Curso

## Instruções

As instruções do projeto do TCC foram dadas em aula e concistem no seguinte

- O trabalho deve ser realizado pelo grupo de TCC já formado.
- Para a avaliação mensal, o grupo deverá entregar um documento com a formatação discutida em aula, sendo ela:
  - Considerações Iniciais
  - Projeto
  - Conclusão

Cada sessão do documento deverá conter o conteúdo descrito nos itens abaixo.

### Considerações Iniciais

Nessa sessão o grupo deverá contextualizar o trabalho a ser desenvolvido, justificando a importância do mesmo. A apresentação do trabalho deverá conter o problema, a proposta de solução e as tecnologias a serem utilizadas.

Leve em consideração colocar nessa sessão de forma sucinta de onde surgiu a ideia do trabalho, quais os problemas que pretendem resolver e como o seu trabalho ajudará a solucionar esses problemas.

Caso utilizem no texto expressões ou termos específicos da área de conhecimento, esses deverão ser definidos no texto. Por exemplo, suponha que o grupo esteja desenvolvendo uma aplicação para determinar, usando uma ferramenta de machine learning, o "valuation" de uma empresa. Nesse caso, o grupo deverá definir o que é valuation, como ele é determinado e quais os métodos disponíveis para isso. Um exemplo de como esse projeto deveria ser apresentado é:

> O valuation de uma empresa é um dos principais indicadores de seu valor de mercado. Ele é determinado por métodos matemáticos e estatísticos que utilizam dados passados e projetados para estimar o valor presente de uma empresa. Os métodos mais comuns para determinar o valuation de uma empresa são o Fluxo de Caixa Descontado (DCF), o Método Comparativo de Empresas (MCE) e o Método dos Prêmios de Mercado (MPM). Para determinar o valuation de uma empresa, é necessário um trabalho manual de coleta, organização e cálculo para cada uma dessas metodologias. Com essa problemática em mente, a empresa Valora, uma empresa de consultoria em valuation, desenvolveu uma aplicação web para automatizar o processo de valuation de empresas.

### Projeto

Nessa sessão o grupo deverá apresentar o projeto a ser desenvolvido, incluindo o que será desenvolvido, as tecnologias a serem utilizadas e a estrutura do projeto. Além disso, nessa sessão deverá ser apresentado o nome da empresa que vocês criaram para o seu trabalho, o slogan, a logo, o nome do produto, e as cores escolhidas para a aplicação que irão desenvolver.

Nessa sessão também, se possível, já apresentem o máximo de detalhes do projeto. Por exemplo:

- Protótipo de telas: Mostrem um protótipo de telas (não precisa ser um protótipo de alta fidelidade, mas que dê uma idéia geral do que será desenvolvido).
- Casos de uso da aplicação: Casos de uso são as principais funcionalidades da aplicação. Eles devem ser apresentados de forma sucinta e clara, com o objetivo de facilitar a compreensão do projeto.
- Diagrama Entidade Relacionamento (ER): O diagrama ER é uma representação visual das entidades e relacionamentos entre elas no sistema. Ele ajuda a entender a estrutura do banco de dados e como as entidades se relacionam entre si.
- Diagrama de Classes: O diagrama de classes é uma representação visual das classes e seus relacionamentos no sistema. Ele ajuda a entender a estrutura do sistema e como as classes se relacionam entre si.
- Diagrama de Sequência: O diagrama de sequência é uma representação visual das sequências de interação entre os objetos no sistema. Ele ajuda a entender a lógica do sistema e como os objetos se comunicam entre si.
- Diagrama de Arquitetura: O diagrama de arquitetura é uma representação visual da arquitetura do sistema. Ele ajuda a entender a estrutura do sistema e como as diferentes partes se comunicam entre si.

Coloquem nessa parte do projeto o máximo de detalhes que vocês já tiverem desenvolvido e comentem sobre as estratégias que vocês pretendem utilizar para a realização do projeto.

### Conclusão

Nessa sessão o grupo deverá apresentar as conclusões do projeto. Deverão ser apresentadas as dificuldades que poderão surgir no decorrer do projeto e como vocês pretendem enfrentá-las.
